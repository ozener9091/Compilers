# Лабораторная работа №1
## Разработка пользовательского интерфейса (GUI) для языкового процессора

### 🎯 Цель работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

**Ситников В.И.**

### 📋 Описание проекта
**GUI для языкового процессора на JavaFX**

Данный проект представляет собой графический интерфейс для языкового процессора, реализованный в виде специализированного текстового редактора. Приложение обеспечивает удобный интерфейс для работы с текстом и базовые функции редактирования.

### 🛠 Используемые технологии
| Технология | Назначение |
|------------|------------|
| **Java 25** | Основной язык программирования |
| **JavaFX 21** | Фреймворк для создания графического интерфейса |
| **IntelliJ IDEA** | Среда разработки |
| **Git** | Система контроля версий |
| **Maven** | Сборка проекта и управление зависимостями |

### 🚀 Инструкция по сборке и запуску

#### 📌 Требования
- Операционная система: Windows / macOS / Linux
- JDK 25 или выше
- Git (опционально)

#### 🔧 Пошаговая инструкция

**1. Установка JDK 25**

Скачайте JDK 25 с [официального сайта Oracle](https://www.oracle.com/java/technologies/downloads/)

**2. Настройка переменных окружения**

*Windows:*
```bash
# Добавьте в PATH:
C:\Program Files\Java\jdk-25\bin

# Проверьте установку:
java -version
javac -version
```

*MacOs/Linux:*
```bash
export JAVA_HOME=/путь/к/jdk-25
export PATH=$JAVA_HOME/bin:$PATH
```

**2. Клонирование и запуск**
```bash
# Клонирование репозитория
git clone https://github.com/your-repo/compilers-laba1.git
cd compilers-laba1

# Компиляция и запуск
javac Launcher.java
java Launcher
```

Запуск через IntelliJ IDEA:

    1. Откройте проект
    2. Настройте JDK 25 (File → Project Structure → SDK)
    З. апустите через Run → Run 'Launcher'

Запуск через Maven:
```bash
mvn clean compile
mvn javafx:run
```
**3. 🖥 Описание интерфейса и функций**
Базовый функционал:

    ✅ Открытие текстовых файлов (.txt)

    ✅ Сохранение файлов

    ✅ Создание нового документа

    ✅ Копирование/Вставка/Вырезание текста

    ✅ Отмена действий

    ✅ Масштабируемые области с ползунками (SplitPane)
